# Jex | Premium Frontend Tools

Jex is a collection of high-performance, premium tools designed for frontend developers to speed up their workflow and add stunning visual elements to their projects with minimal effort.

![Jex Preview](./public/previewjex.png)

## ‚ú® Features

- **Mesh Gradient Generator**: Create beautiful, flowing mesh gradients with customizable control points and export them directly to CSS.
- **Dot Art Generator**: Transform any image into a stylish SVG dot matrix pattern, perfect for data visualization or decorative backgrounds.
- **Bento Grid Generator**: Design responsive Bento-style layouts with an interactive editor and export the code in Tailwind CSS or Vanilla CSS.
- **Animated Backgrounds**: A curated collection of high-performance, CSS-only animated backgrounds ready to copy and paste.
- **CSS Patterns**: A library of modern, drop-in CSS background snippets and layout patterns.

## üöÄ Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm / pnpm / yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/jex.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

### Building for Production

To create an optimized production build:
```bash
npm run build
```
